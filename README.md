# Deduplication

本项目是本人学习文件固定分块去重时写的一个小项目，请确保使用全英文测试...

## Start

1.  首先请将项目克隆至本地

    ```bash
    git clone https://github.com/reagin/Deduplication.git
    ```

2.  在项目路径中打开终端，并安装依赖项

    ```bash
    cd Deduplication; npm install;
    ```

3.  运行本项目，默认打开于`http://127.0.0.1:3000`

    ```bash
    npm run start
    ```
